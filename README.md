# 3Dはくせい図鑑

iPhone LiDARで取得した剥製の3Dモデルを、機種を問わず閲覧できるWebサイト。
八ヶ岳総合博物館との連携プロジェクトとして開発。

## 概要

LiDARを搭載したiPhoneで剥製の3Dスキャンを行い、Web上で誰でも閲覧できるデジタル図鑑を構築。
LiDAR対応iPhoneでしか3Dスキャンを直接体験できないという制約を、
Webサイトでの公開によって機種・OS問わず閲覧可能とした。

## 公開URL

https://yurayura3812-svg.github.io/3d-hakusei-zukan/

## 技術スタック

- HTML / CSS / JavaScript
- model-viewer (3Dモデル表示ライブラリ)
- GitHub Pages (ホスティング)
- iOS / Android で最適な3D形式を出し分け
  - iOS: USDZ
  - Android / その他: GLB

## 機能

- 剥製の3Dモデルをブラウザ上で閲覧
- 機種・OSに応じた最適な3D形式の自動出し分け
- 360度回転、ピンチ拡大縮小、AR表示(対応端末)

## 背景・課題解決

文化財や標本のデジタル保存は重要だが、3Dデータの閲覧には対応端末や専用アプリが必要なことが多い。
本プロジェクトでは、博物館の来場者が「機種を問わず」「ブラウザだけで」3Dモデルを楽しめるよう、
Webサイトでの公開を選択。さらに、機種ごとに最適な形式を自動で配信する仕組みを実装した。
